1. What is DOM and what is the purpose of DOM?

Ans: Dom is Document Object Model. DOM can do a programming interface for web documents. By using DOM we can change website document structure, content and style. It represents the document as nodes and objects. Also we can create applications that update the data of the page without needing a refresh.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: We can select HTML DOM elements by the name attribute using the javascript “getElementsByName()” method. The DOM methods name are given bellow:

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByName()
  4. getElementsByTagName()
  5. querySelector()
  6. querySelectorAll()

3. What is event bubble?

Ans: Event bubble is a type of DOM event propagation.When we use event bubbles, it go up to the child and parent. Event bubble have not a click event listener. It use for adding individual event listeners to each child element.

4. What is a callback function and why will you use it?

Ans: Callback function passed into another function as an argument and called bt the function when we set event condition. Callback function use for event handling. For example we set a button and it given a callback function then when we click the button function, the function can be occurs event occurs.
